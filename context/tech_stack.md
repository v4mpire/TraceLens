Frontend:
- Next.js (TypeScript)
- Tailwind CSS
- Framer Motion

Backend:
- Node.js (TypeScript)
- OpenTelemetry SDK and collector
- REST ingestion APIs

SDKs:
- @tracelens/web for browser instrumentation
- @tracelens/node for server instrumentation

Data:
- PostgreSQL for traces, graphs, CVEs
- Object storage for raw telemetry

Infrastructure:
- Dockerized services
- Self-hosted using Coolify on a VPS
